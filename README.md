# Treinoway Landing Page

Landing page simples para apresentar a Treinoway, plataforma que conecta alunos e treinadores para treinos personalizados.

## Tecnologias

- HTML5
- CSS3 (estilos embutidos em `index.html`)
- JavaScript puro (apenas para o menu mobile)

## Como visualizar

1. Certifique-se de que o arquivo `index.html` e as imagens (`treino_way_transparente.png`, `treinoway.ico`) estão na mesma pasta.
2. Abra o arquivo `index.html` diretamente no navegador (arrastando o arquivo para a janela do navegador ou com “Abrir arquivo”).

## Estrutura

- `index.html` – página principal da landing page.
- `README.md` – este arquivo de descrição do projeto.

## SEO
